# CS61A

这是一个入门级程序设计课程。相比于国内的程序设计课而言，其更会让人理解程序为什么会这样运行，而不只是教授如何写出程序。课程非常重视函数抽象、递归、作用域、程序求值过程等内容，因此其整体难度并不低。无论是前半部分的程序设计内容，还是后面使用的 Scheme 和 SQL，对初学者来说都有一定的学习门槛。

课程前面会使用 python 讲解程序设计的内容，包括高阶函数、递归、数据抽象、面向对象、迭代器和生成器等内容，会让你熟悉程序设计，对程序结构和运行都有。之后会学习与使用 Scheme 和 SQL 语言，更会引导你使用 Python 实现 Scheme 解释器，这部分也是这门课程极具吸引人的地方。

可以选择看视频或者看课程的教材，若教材很难看懂，老师的视频讲解会更好懂一些，各种内容讲的都很清晰。课程会设置 lab，homework 和 project 这三个练习，discussion 部分也可以了解一下，这些都是课程的精华，应该重点完成。对于 project，会给出项目的框架，引导你写出几个特定的部分，但是有些题目描述比较长，也比较抽象，需要反复阅读代码框架和测试用例才能真正理解要求。

总体来说，这是一个很好的入门程序设计的课程，会对程序的组织和运行形成更加系统的理解。
